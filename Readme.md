# Atuhaire Abiudi Portfolio Website

## Prerequisites
- Docker
- Docker Compose (optional)

## Local Development
1. Build Docker image:
   \`\`\`bash
   docker build -t portfolio-website .
   \`\`\`

2. Run container:
   \`\`\`bash
   docker run -p 8080:80 portfolio-website
   \`\`\`

3. Access website:
   Open browser to http://localhost:8080
